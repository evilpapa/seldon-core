一个可大规模地在 Kubernetes 上部署机器学习模型的开源平台。

概述
--------

Seldon Core 将你的机器学习模型（Tensorflow, Pytorch, H2o等）或（Python, Java等）语言封装转换为生产环境 REST/GRPC 微服务。

Seldon 处理着成千上万的生产环境机器学习模型实例，并提供先进的开箱即用的能力，包括高级指标，访问记录，解释器，异常值检测，AB测试，金丝雀发布等特性。
